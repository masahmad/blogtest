notes 

a basic angularjs blog app in a single page.  

temp folder contains backend nodejs script

key points:
ui-router for partial / nested views. 
$resource factory for restful services


blog create
-----------
basic blog creation (heading / content and keyword tag form fields
basic form validation ( heading required)


blog listing
------------
listing of blog heading with filtering available based on tags.
Example: a fruit blog can be filtered by typing 'apples' 


blog detail
-----------
displays blog detail



other considerations/ notes.
--------------------
there is no caching of http services - this would be recommended


backend
-------
built with nodejs / sql server - scripts included in temp folder
